# HARS — Comparative Analysis Project

Cybersecurity capstone project evaluating the Household Adaptive Risk-Scoring (HARS)
framework against existing smart home IoT security applications.

## Assignment Focus

This project's deliverable is a **comparative analysis** — evaluating HARS against
three existing, real-world applications:

- **Fing** — consumer network-discovery and device-scanning app
- **SecureIoT.house Smart Home Security Scorecard** — consumer browser-based risk
  assessment tool
- **IoT Inspector** — free, open-source academic research application

The comparison uses six evidence-based criteria drawn directly from HARS's own design
requirements (Part A, Table 6/7): household usability, weighted composite scoring,
evidence traceability, lightweight/no-cloud deployment, device heterogeneity coverage,
and cost.
